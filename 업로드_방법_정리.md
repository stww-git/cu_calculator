# GitHub에 파일 업로드하는 방법

## 방법 1: GitHub 웹에서 직접 파일 생성 (권장) ⭐

`.github` 폴더는 숨김 폴더이므로, GitHub 웹에서 직접 생성하는 것이 가장 쉽습니다.

### 단계:
1. GitHub 저장소 페이지에서 "Add file" → "Create new file"
2. 파일 경로 입력: `.github/workflows/build_windows.yml`
3. 파일 내용 복사해서 붙여넣기
4. 저장

## 방법 2: 폴더 전체 업로드

### 필요한 파일들:
- `calculator_cu.py` (필수)
- `.github/workflows/build_windows.yml` (필수)

### 업로드 방법:

#### A. GitHub 웹에서 업로드
1. 저장소 페이지에서 "uploading an existing file" 클릭
2. `calculator_cu.py` 파일 드래그 앤 드롭
3. `.github` 폴더는 숨김 폴더라서 직접 업로드가 어려울 수 있음
   → 그래서 방법 1(웹에서 직접 생성)을 권장합니다!

#### B. Git 명령어 사용 (터미널)
```bash
# 저장소 클론 (또는 초기화)
git init
git remote add origin https://github.com/사용자명/저장소명.git

# 파일 추가
git add calculator_cu.py
git add .github/workflows/build_windows.yml

# 커밋 및 푸시
git commit -m "Add CU 발주시스템"
git push -u origin main
```

## 방법 3: 압축 파일로 업로드 후 GitHub에서 추출

1. `calculator_cu.py`와 `.github` 폴더를 압축
2. GitHub에서 압축 파일 업로드
3. GitHub가 자동으로 압축 해제 (일부 경우)

## ⚠️ 중요 사항

- `.github` 폴더는 반드시 필요합니다 (워크플로우 파일이 들어있음)
- `.github` 앞의 점(.)이 중요합니다 (숨김 폴더 표시)
- 파일 경로: `.github/workflows/build_windows.yml` (정확히 이 경로여야 함)

## ✅ 최종 확인

업로드 후 확인:
1. 저장소에서 `.github/workflows/build_windows.yml` 파일이 보여야 함
2. Actions 탭에서 "Build Windows Executable" 워크플로우가 보여야 함
