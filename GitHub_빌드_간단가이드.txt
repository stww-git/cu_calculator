========================================
  GitHub Actions로 Windows 실행 파일 만들기
  (Windows PC 없이 가능!)
========================================

🎯 4단계로 완성!

1단계: GitHub 계정 만들기
----------------------------------------
- https://github.com 접속
- 회원가입 (무료)

2단계: 새 저장소 만들기
----------------------------------------
- 우측 상단 "+" → "New repository"
- 이름 입력 후 "Create repository"

3단계: 파일 업로드
----------------------------------------
필요한 파일:
- calculator_cu.py
- .github/workflows/build_windows.yml

업로드 방법:
- 저장소 페이지에서 "uploading an existing file" 클릭
- 파일 드래그 앤 드롭
- "Commit changes" 클릭

4단계: 빌드 실행 및 다운로드
----------------------------------------
- "Actions" 탭 클릭
- "Build Windows Executable" 선택
- "Run workflow" 클릭
- 완료 후 Artifact에서 .exe 파일 다운로드
