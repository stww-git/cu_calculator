========================================
  CU 발주시스템 사용 설명서 (Windows)
========================================

📦 프로그램 정보
- 프로그램명: CU발주시스템.exe
- 버전: 1.0
- 용도: 편의점 자동 발주 계산 시스템

🚀 실행 방법

1. "CU발주시스템.exe" 파일을 더블클릭하세요.
2. 명령 프롬프트 창이 열리며 프로그램이 실행됩니다.

⚠️  Windows 보안 경고 해결 방법

처음 실행 시 Windows Defender나 백신 프로그램이 경고를 표시할 수 있습니다.

해결 방법:
1. "추가 정보" 또는 "자세히" 클릭
2. "실행" 또는 "허용" 버튼 클릭
3. 또는 Windows Defender에서 예외 추가:
   설정 → 업데이트 및 보안 → Windows 보안 → 
   바이러스 및 위협 방지 → 설정 관리 → 
   제외 추가 또는 제거 → 파일 추가

💡 사용 방법

1. 프로그램을 실행하면 발주 마감일과 참고일이 표시됩니다.
2. 참고일의 판매량을 입력하세요.
3. 입력값을 확인한 후 발주 결과를 확인하세요.

📋 입력 항목
- 도시락
- 삼각김밥
- 김밥
- 샌드위치
- 햄버거
- 빵
- 디저트

📊 출력 결과
- 2차 발주 품목: 1차/2차 발주량 표시
- 1차 발주 품목: 총 발주량 표시
- 총 발주량 요약

🔧 시스템 요구사항
- Windows 7 이상
- 별도의 Python 설치 불필요
- 별도의 추가 프로그램 설치 불필요

❓ 문제 해결

프로그램이 실행되지 않으면:
1. 관리자 권한으로 실행해보세요
2. 백신 프로그램이 차단하는지 확인하세요
3. Windows 버전 확인 (Windows 7 이상 필요)

명령 프롬프트 창이 바로 닫히면:
- 프로그램을 명령 프롬프트에서 실행:
  cmd를 열고 프로그램이 있는 폴더로 이동한 후
  CU발주시스템.exe 입력
