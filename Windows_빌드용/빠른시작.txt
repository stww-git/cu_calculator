========================================
  빠른 시작 가이드
========================================

🚀 3단계로 실행 파일 만들기

1단계: Python 설치 확인
----------------------------------------
- Python이 설치되어 있지 않다면:
  https://www.python.org/downloads/ 에서 다운로드
  ⚠️ 설치 시 "Add Python to PATH" 반드시 체크!

2단계: 빌드 실행
----------------------------------------
- "build_windows.bat" 파일을 더블클릭
- 자동으로 빌드가 진행됩니다
- 완료되면 Enter 키를 누르세요

3단계: 실행 파일 확인
----------------------------------------
- "dist" 폴더가 생성됩니다
- "dist\CU발주시스템.exe" 파일이 생성되었습니다
- 이 파일을 배포하세요!

========================================
💡 팁
- 빌드가 완료되면 dist 폴더의 .exe 파일을
  Windows_사용설명서.txt와 함께 압축하여 배포하세요.
